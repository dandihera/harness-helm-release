# harness-helm-release

harness-helm 배포 채널. 설치 스크립트와 runtime asset을 제공한다.

## 설치 (macOS / Linux)

```sh
sh -c "$(curl -fsSL https://raw.githubusercontent.com/dandihera/harness-helm-release/main/install.sh)"
```

## 설치 (Windows PowerShell)

```powershell
irm https://raw.githubusercontent.com/dandihera/harness-helm-release/main/install.ps1 | iex
```

## 업데이트

```sh
sh -c "$(curl -fsSL https://raw.githubusercontent.com/dandihera/harness-helm-release/main/install.sh)"
```
